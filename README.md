1. [Create a repo](https://github.com/itw-creative-works/mobile-app-manager-template/generate) from the **Mobile App Manager Template** template.
2. Clone the repo to your local machine.
3. Run these command to get everything setup and sync'd!

```bash
# or

```
4. Start building and testing your mobile app!
